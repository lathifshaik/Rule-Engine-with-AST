Rule Engine

Create Rule
Instructions: Enter a name for your rule and the rule string. The rule string should be a valid logical expression using operators like AND, OR and parentheses. Note: The rule engine is case-sensitive.
Evaluate Rule
Instructions: Select a rule to evaluate and enter the JSON data to test against the rule. Ensure your JSON is properly formatted and includes all required fields. The evaluation is case-sensitive.
Combine Rules
Instructions: Enter the IDs of the rules you want to combine, separated by commas. Choose the operator (AND/OR) to combine the rules. The combination is case-sensitive.
All Rules
    Instructions: Click the "Fetch Rules" button to retrieve and display all existing rules. Click the "Edit" button to modify a rule. Rule names and strings are case-sensitive.
